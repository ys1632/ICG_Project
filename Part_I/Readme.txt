#How to run

1.Create a new Win32 Console Application project in VC++.
2.Paste my source code into the empty project.
3.Put input.txt under the same direcotry with new project source code.
4.Add glew32.lib;glut32.lib;opengl32.lib;glu32.lib; into additional dependencies under configuration properties.
5.Click on Debug to run it.
